Game of Life

The Game of Life

The game of life is a game created by John Conway. It is a simulator for a given neighborhood with a set of rules for each "cycle", a cycle being checked all spaces on the current board. The rules being:
    For each space that already has a home:
  • If the space has 0 to 1 neighbors, it dies by starvation
  • If the space has four or more neighbors, it dies by overpopulation
  • If the has 2-3 neighbors, it survives
    • For each space that has no home on it:
  • It becomes populated if it has exactly three neighbors

How it Works

The most recent version of the game is a web-app implementation using javascript where I took the original idea from JAVAFX and re-implemented it with a new recursive algorithm and added many features not found on other game of life simulators. In short, I added zooming in and out of a designated area of the map, allowing to drag the map around to have a better viewing experience, and the ability to change the framerate of the simulation. I had to make sure the dragging and zooming had to provide the correct reaction depending on the zoom of the current user (the size of the cell). This was done by taking the size of the cell and finding where the cell was with respect to the center of the board so that I could render the cell in the correct position after the user drags or zooms. The creation of the cells by the user was also done using a similar approach since I would need to get the cell at where the user clicked, and find it with respect to the center of the board. Since the cell's could be at any given length, and the board could have been dragged anywhere by the user, I would need to take into account the length of the cell and the relative coordinates of the cell with respect to both the page, and the board.

Spam E-Mail Classifier

What is a Spam Classifier?

A spam classifier is an algorithm that takes a body of text and determines if that body of text resembles a "spam" like term. This "spam" term can be any attribute that we the classifier to look for based on the training.

How it works

This classifier works by using the popular Naive Bayes Algorithm as a default implementation and then adding a part-of-speech(POS) weight to the algorithm to determine the reliability of introducing such a weight. Buzzword

The Game of Life

Buzzword is a word based grid game. The player makes words that fit the criteria to proceed score points and proceed to later levels.

How it Works

Buzzword features a simple encrypted login system, many levels, and a scaleable board.

    Frameworks:
  • The board is generated though a framework which handles it's creation along with the event handlers associated with the board. The user asks the framework for a board and the framework can provide a board with variable specifications and deals with events on a seperate thread.
  • Nearly every part of the app is created using a framework which regulates the colors and size of each object, therefore allowing the user of the framework an easy way to change many portions of the app.

    • The GUI:
  • The GUI was created using JavaFX and the app itself was created using a app framework provided.
